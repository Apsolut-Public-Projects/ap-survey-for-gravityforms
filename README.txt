== Description ==

Activate GravityForms and Install Survey Add-on
- After that you can Install and Activate AP Survery for GravityForms,
 please use ready class names so CSS and JS files are added to form (Look into How to use it section)


== Installation ==


1. Upload `ap-survey-for-gravityforms.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to use it =
1. Please add class name `ap-survey-for-gf` to Form > Settings > CSS Class Name
2. Add conditional buttton (checkbox, radio) and use class name `ap-survey-conditional`,
   this will automatical move button after submit button (so its in the same place)
